Developer Guide
1. The Data Utility (dataHandler.js)
To interact with our JSON "database," always import the utility at the top of your controller:

const { readData, writeData, generateId } = require('../utils/dataHandler');
2. How to Perform CRUD Operations
DO NOT use fs directly. Use these standardized patterns to maintain data integrity:

Read Data
To fetch the entire array from a specific JSON file:

const users = readData('users');
Create Data
To add a new entry, read the existing array, push the new object, and save the entire array back to the file:

const jobs = readData('jobs');
const newJob = { 
    id: generateId('JOB'), 
    ...req.body 
};
jobs.push(newJob);
writeData('jobs', jobs);
Update Data
To modify an existing entry, find its index in the array, update the specific fields, and save:

const apps = readData('applications');
const index = apps.findIndex(a => a.id === targetId);

if (index !== -1) {
    apps[index].status = 'shortlisted';
    writeData('applications', apps);
}
